## Filosofía del Repositorio

Este repositorio sigue el principio de **honestidad epistémica radical**.

- El núcleo científico (Nivel 1-2) se mantiene limpio y reproducible.
- El material exploratorio y especulativo (Nivel 5-6) se separa claramente.
- Lo que fue falsado se archiva con transparencia, no se borra.

La TCU no es solo una teoría sobre mercados o sistemas complejos. Es un **marco para diseñar sistemas coherentes** — incluyendo sistemas de inteligencia artificial soberana.

## TCU como Arquitectura para Sistemas Soberanos

La Teoría de Coherencia Unificada puede (y debe) usarse como **capa meta-arquitectónica** para el diseño de sistemas de IA soberana:

- **Memoria persistente** como kernel de memoria exponencial (κ·I[C])
- **Activación de skills** cuando se supera un umbral de coherencia (Q > Q_c)
- **Resonancia entre agentes** como sincronización de fase
- **Detección de crisis sistémicas** (SHOCK vs MACRO) en el propio ecosistema de agentes
- **Poda adaptativa** de skills y conexiones que generan entropía

Este repositorio no solo contiene la teoría. Contiene el **código fuente conceptual** para construir sistemas más coherentes.
