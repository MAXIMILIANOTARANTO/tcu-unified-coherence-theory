## 3. Núcleo Sólido Actual (Nivel 1-2)

### 3.1 Ecuación Maestra (consolidada)

```
□C + μ^{2}C + λ|C|^{2}C + κ·I[C] = S(x,t)
```

Donde el operador de memoria explícito es:

```
I[C](x,t) = ∫_{-∞}^t (1/τ_m) · exp(-(t-s)/τ_m) · C(x,s) ds
```

Este formalismo es internamente consistente y pertenece a la familia de ecuaciones de Ginzburg-Landau con memoria exponencial causal.

### 3.2 Hallazgos Empíricos Confirmados (Paper v3 / Experimentos 1-8)

| Hallazgo | Evidencia | Nivel |
|----------|---------|-------|
| Absorption Ratio (AR1) captura colapso de coherencia | AUC = 0.818, p = 0.036 vs VIX-only | Nivel 1 |
| Critical Slowing Down (CSD) detecta crisis macro | AUC_macro = 0.832 | Nivel 1 |
| Taxonomía bimodal de crisis: **SHOCK vs MACRO** | p = 0.007 (separación clara) | Nivel 1 |
| τ_m^field (calculado sobre AR1) muestra comportamiento direccional opuesto | p = 0.016 | Nivel 2 |
| AR1 ≈ Q (parámetro de orden) | r = 0.80 | Nivel 2 |
| Hilbert sobre retornos diarios financieros | Falsado correctamente (no funciona) | Archivado |

### 3.3 Taxonomía de Crisis (confirmada)

- **SHOCK (exógeno):** S domina. τ_m disminuye antes del evento. Recuperación más rápida.
- **MACRO (endógeno):** κ·I[C] domina. τ_m aumenta antes del evento (acumulación de memoria). Recuperación más lenta y compleja.

Esta taxonomía tiene valor predictivo y diagnóstico real.
